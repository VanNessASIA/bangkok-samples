Counter / late-night mood

一人でも入りやすく、
二軒目でも帰りたくない。

カウンターは“ただの席数”ではなく、ブランドそのもの。 焼き台の手元が見えること、グラスの音が届くこと、会話が少し漏れること。 その近さが、深夜帯の再訪理由になります。

カウンター 12席 焼き場を正面にした主役席。炭の表情と手仕事がそのまま演出になる配置。
奥の小上がり 16席 会社帰りの少人数グループにちょうどいい密度。騒がしすぎず、静かすぎない空気感。
深夜の照明設計 明るく見せるのではなく、顔色がよく見える低い暖色。提灯と木肌が主役になる照度に調整。
Sake / highball / first drink

ハイボールは軽く、
日本酒は温度で魅せる。

このタイプではドリンクも“映える棚”ではなく、料理を押し上げる役回り。 最初の一杯の爽快感と、終盤に向けた日本酒の深さを両方置ける構成が似合います。

角ハイボール 炭火の脂をすっと切る、細かい泡の一杯。最初の注文率が高い看板ドリンク。
赤星 中瓶 横丁感に合う、古びたラベルの説得力。焼き鳥の塩と最短距離でつながる組み合わせ。
燗で出す純米酒 冷酒だけで終わらせず、深夜帯に温度のある提案を入れることで、店の記憶が残りやすくなります。
Access / reservation

トンローの中で、
少しだけ奥まった一軒へ。

路地の空気を守るため、表通りに開きすぎない入口設計を想定。 その代わり、見つけた時に“当たりの店っぽい”と思わせる提灯と炭の香りを強く残します。
